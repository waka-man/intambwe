# Intambwe

*Designing Africa's Next Icons.*

Intambwe is a multidisciplinary creative studio and design house based in
Kigali, Rwanda. We create distinctive brands, immersive interiors, bespoke
furniture, lighting, and visual experiences that blend African creativity with
contemporary global design.

Every project begins with a story and develops into a cohesive design system
that extends across identity, space, products, and culture — purposeful,
memorable, and built to endure.
